## 🔄 方案对比

| 方面 | IRSA 方案 (已废弃) | AK/SK 方案 (当前) |
|------|-------------------|-------------------|
| **凭证类型** | 临时凭证 (12小时有效期) | 长期静态凭证 |
| **配置复杂度** | 高（需要 ServiceAccount、IAM Role） | 低（只需环境变量） |
| **凭证刷新** | 需要后台线程刷新 | 无需刷新 |
| **内存占用** | 较高（有后台线程） | 较低 |
| **稳定性** | 可能因刷新失败导致问题 | 更稳定 |
| **调试难度** | 困难（credential chain 复杂） | 简单 |
| **推荐使用** | ❌ 不推荐 | ✅ 推荐 |
